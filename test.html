Setup Instructions:

  1. Open the ReqCraft extension popup
  2. Add a new rule:
    • Name: Test Rule
    • URL Pattern: https://httpbin.org/*
    • Method: ALL
    • Header: Add header X-Test-Header with value it-works
  3. Click the test buttons below
  4. Check if the custom header appears in the response

Test 1: Simple GET Request

This will make a GET request to httpbin.org/headers which echoes back all headers it receives.

Test 2: POST Request with Body

This will make a POST request to httpbin.org/post.

Test 3: Query Parameter Test

Test adding query parameters to the request.
